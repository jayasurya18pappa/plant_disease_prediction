Plant Disease Prediction using ML

Overview

This project predicts plant diseases using machine learning algorithms:

* Random Forest
* Support Vector Machine (SVM)

Dataset
 Dataset used:
 New Plant Diseases Dataset (Kaggle)
 We used a smaller subset with 3 classes:
   * Early Blight
   * Late Blight
   * Healthy

Technologies Used

  * Python
  * OpenCV
  * Scikit-learn
  * NumPy

How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the program:
   python main.py

Models Used

  Random Forest

* Ensemble learning method
* Works well with image features

SVM

* Effective for classification
* Works well on small datasets

Results

The models are compared based on accuracy:

* Random Forest Accuracy: %40
* SVM Accuracy: %61
